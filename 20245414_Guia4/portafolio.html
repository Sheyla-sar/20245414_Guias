Descripción del curso

Materia: desarrollo web

En el curso de Desarrollo Web I, he aprendido sobre fundamentos del diseño y la construccion de sitios web utilizando distintas tecnologías, entre ellas: HTML 5, CSS 3 Y JavaScript. Por otro lado, también realizamos guías practicas individuales, utilizando herramientas como Vissual Studio Code, navegadores web y GitHub Pages para la publicación de proyectos.

Competencias aprendidas:

  • Comprender y utilizar los diferentes elementos de HTML para elaborar la estructura de la aplicación web
  • Comprender las propiedades y selectores utilizados en las hojas de estilo en cascada para aportar elementos de estilo a una aplicación web
  • Desarrollar elementos que permitan manejar la interacción del sistema con el usuario de acuerdo con elementos de sintaxis moderna en Javascript o superconjuntos de dicho lenguaje

Contenidos vistos:

  • UNIDAD I: Introducción al HTML
  • UNIDAD II: Introducción a las hojas de estilo en cascada (css)
  • UNIDAD III: Maquetación de sitios web
  • UNIDAD IV: Frameworks de diseño responsivo
  • UNIDAD V: Introducción a JavaScript
  • UNIDAD VI: El modelo de objetos del documento (DOM) y JavaScript asíncrono
